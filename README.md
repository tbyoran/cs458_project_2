# CS458 Project #2

## Install

## Run
```bash
  npm start
```
```bash
  npm run android
  npm run ios
```

## Test
```bash
  npm run test
```

## Build
```bash
  npm run build:android
```
After the build script is done the generated apk (`app-release.apk`) can be found in `./android/app/build/outputs/apk/release`.
